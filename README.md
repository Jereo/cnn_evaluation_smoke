# Pretrained CNN Models on Smoke Dataset

Training parameters:
- 10 batches
- 100 epochs
- RMSProp optimizer (learning rate = 0.001)
- Training and evaluation sets: http://staff.ustc.edu.cn/~yfn/vsd.html (Sets 1 to 4)
